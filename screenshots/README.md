## Cara Mengambil Screenshot

1. Jalankan aplikasi: `php artisan serve`
2. Buka browser dan akses: `http://localhost:8000`
3. Login sebagai admin atau student
4. Navigasi ke halaman yang ingin di-screenshot
5. Gunakan tool screenshot:
   - Windows: `Win + Shift + S`
   - Mac: `Cmd + Shift + 4`
   - Browser: Extension Full Page Screen Capture
6. Simpan dengan nama sesuai daftar di atas

## Tips

- Gunakan resolusi tinggi (min 1920x1080)
- Pastikan semua elemen terlihat jelas
- Gunakan mode fullscreen untuk hasil terbaik
- Pastikan data dummy terlihat jelas (jika ada)

Lihat `PANDUAN-SCREENSHOT.md` di root folder untuk penjelasan detail setiap screenshot.
